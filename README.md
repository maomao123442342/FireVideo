A Video Player using QT5 and VS2013

主要功能：                                                                                                                                
支持拖拽添加多条视频，自动识别文件类型 																																																			
支持右键功能选项，可自主设定视频播放顺序 																																																		
支持屏幕单击暂停/播放，双击全屏模式 																																																		
双击视频列表可播放当前视频 																																																		
支持全屏模式，全屏界面下的功能选项等待后期更新
